# Hero WAR

## Description
Application Vue 3 qui permet de rechercher des super-heros, en selectionner 2, puis lancer un duel tour par tour avec des stats calculees depuis SuperHeroAPI.

## Configuration `.env`
Copie le fichier d'exemple puis remplace le token :

```sh
cp .env.dist .env
```

Dans `.env` :

```env
VITE_SUPERHERO_TOKEN=INSERT_YOUR_TOKEN_HERE
```

Remplace `INSERT_YOUR_TOKEN_HERE` par ton token SuperHeroAPI (https://superheroapi.com/)

## Project setup

```sh
npm install
npm run dev
```
